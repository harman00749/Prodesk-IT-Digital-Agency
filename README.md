Prodesk IT Digital Agency Landing Page

Overview
A responsive landing page for the fictional agency "Prodesk IT". Built with HTML, CSS, and JavaScript, it demonstrates a professional design with dark mode support and mobile compatibility.

Features
- Sticky navbar with hamburger menu
- Hero section with "Get Started" button
- Full-page dark mode toggle
- Services section (SEO 🔍, Web Development 💻, Digital Marketing 📈) with emojis and blue headings in dark mode
- Hover animations for buttons and cards
- Responsive grid layout for all screen sizes
- Footer with social icons

Technologies
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (dark mode toggle, hamburger menu)
